How to contribute?

Before starting the contribution, make a Fork in the project.

  • $ git clone https://github.com/iagocavalcante/qtube.git
  • $ cd qtube
  • $ npm install
  • # The construction process can be time consuming due to the download of the Quasar and Electron
  • $ npm run build
